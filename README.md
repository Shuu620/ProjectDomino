## ✨ Project Overview

이 프로젝트는 **도미노피자 코리아 웹사이트의 UI/UX를 학습용으로 재현한 클론 프로젝트**입니다.

반응형 헤더와 네비게이션, 배너 슬라이더, 메뉴 페이지, 매장검색, 이벤트/제휴/쿠폰 관련 화면을 구현하며  
실서비스에 가까운 구조를 직접 구성해보는 데 중점을 두었습니다.

## 🚀 Main Features

- 반응형 헤더 및 GNB 메뉴 구성
- 스크롤 고정(Sticky) 헤더와 드롭다운 메뉴 구현
- 메인 배너 슬라이더 및 프로모션 영역 구성
- 메뉴 페이지 및 카드형 리스트 UI 구현
- 카카오 지도 API를 활용한 매장검색 기능
- 이벤트 / 제휴 / 쿠폰 / 기프트 관련 정적 페이지 구성

<br>

## 🧩 What I Did

- 실제 브랜드 사이트 흐름을 참고한 화면 설계
- 메인 페이지 및 서브 페이지 퍼블리싱
- 공통 스타일 및 스크립트 구조 정리
- 메뉴/프로모션/이벤트 중심 UI 구현
- 외부 라이브러리와 API를 활용한 사용자 경험 확장

<br>

## 📄 Page Structure

- `index.html` : 메인 페이지
- `menu.html` : 메뉴 페이지
- `store_search.html` : 매장검색
- `eventAff_event.html` : 이벤트
- `eventAff_Affiliation.html` : 제휴 / 할인
- `ecoupon_order.html` : E-쿠폰 주문
- `ecoupon_instruction.html` : E-쿠폰 이용안내
- `gift.html` : e-기프트
- `login.html` : 로그인
- `sign_up.html` : 회원가입
- `notification_news.html` : 공지 / 뉴스
- `franchise.html` : 가맹 / 창업
- `costomer_compliment.html` : 고객센터
- `header.html`, `footer.html` : 공통 컴포넌트 샘플

### ▶️ How to Run
1. Clone repository<br>```
git clone https://github.com/Shuu620/ProjectDomino.git```
2. Move to project folder<br>```
cd ProjectDomino```
3. Open in browser<br>
- `index.html` 파일을 브라우저에서 실행
또는
- VSCode Live Server로 실행
